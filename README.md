# Slash UP

## 作品概要

プレイヤーがキューブを操作し、斬撃や必殺技を駆使しながら敵を倒していくアクションゲームです。
敵を倒して強化を行いながらステージ攻略をしていきます。

## 制作情報
・開発人数：7人
・制作期間：3か月
・開発環境：Unity/C#
・制作形態：チーム制作

## 担当箇所

私は以下の実装を担当しました

・攻撃システム
・エフェクト制作
・システム統合
・デバック
