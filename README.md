# Slash

## 作品概要

プレイヤーがキューブを操作し、斬撃や必殺技を駆使しながら敵を倒していくアクションゲームです。
敵を倒して強化を行いながらステージ攻略をしていきます。

## 制作情報
・開発人数：7人
・制作期間：3か月
・開発環境：Unity/C#
・制作形態：チーム制作

## 担当箇所

私は以下の実装を担当しました

・プレイヤー制御
・攻撃システム
・エフェクト制作
・システム統合
・デバック

## 担当内容詳細

### プレイヤー制御

プレイヤーの移動、ジャンプ、被ダメージ、無敵時間、死亡処理を実装しました
操作時の挙動やレスポンスを調整し、快適に遊べる操作感を目指しました

### 攻撃システム

通常攻撃では斬撃オブジェクトを生成し、スキル取得によって攻撃範囲が変化する仕組みを構成しました。
飛び道具攻撃では単発攻撃だけではなく、3方向攻撃や5方向攻撃などの複数攻撃の攻撃パターンを実装しました。
必殺技では専用ゲージが満タンになると発動可能となり、画面演出と連動して敵へ大ダメージを与えるシステム
を制作しました。

### エフェクト制作
斬撃エフェクトやシールド破損エフェクト、死亡エフェクトなどを制作しました。攻撃時の爽快感
を演出できるよう調整を行いました。

### システム統合
チームメンバーが制作した敵システムやスキルツリーシステムをゲームへ統合しました。
機能同士の連携確認や不具合修正を行い、安定して動作するよう改善しました。

## 工夫した点

スキル取得によって攻撃パターンや性能が変化する仕組みを導入し、
プレイヤーが成長を実感できるよう工夫しました。また、チーム開発では
他メンバーのコードを確認しながら統合作業を行い、システム間の
不具合解消に取り組みました。

## 学んだこと
チーム開発を通じて、自分以外が作成したコードをを理解しながら開発を進める
経験を積むことができました。また、複数のシステムを統合して一つのゲーム
として完成させる難しさと重要性を学びました。
